reportLastUpdated method

dynamic reportLastUpdated ()

Retrieves the latest 'lastBuildDate', which is meant to be the last time the reportResource report was updated. Note, this will return null if the cache is empty (meaning latest was never called)

Implementation 

reportLastUpdated() => new DateFormat("EEE, dd MMM yyyy hh:mm:ss zzz")
    .parse(_cache?.response.findAllElements("lastBuildDate").first.text);
