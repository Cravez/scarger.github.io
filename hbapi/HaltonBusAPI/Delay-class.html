Delay class

Wraps items from https://haltonbus.ca/ bus delays in an obscure-free and simple manner. Route, Schools, Time and status may be accessed through this API

Constructors

Delay(String delayData)
Build a delay object, wrapping delayData as the payload from an HTTP request to the site
